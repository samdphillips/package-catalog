Demo Racket Package Catalog

This is an example showing how to use GitHub Pages to host a Racket Package Catalog.

How to use this catalog

$ raco pkg config --set catalogs \
    https://samdphillips.github.io/pkg-catalog-demo/catalog/ \
    https://download.racket-lang.org/releases/8.4/catalog/   \
    https://pkgs.racket-lang.org                             \
    https://planet-compats.racket-lang.org

# Check the catalog contents.  This is using a URL that is pinned to the tag
# v0.10.0.  The same package reference on pkgs.racket-lang.org is pointing to
# the release branch.
$ raco pkg catalog-show keyring
Package name: keyring
 Source: https://github.com/samdphillips/racket-keyring.git?path=keyring#v0.10.0
 Checksum: b94d316071f10b14c32bf7398d45d849348e680d

Notes

These are things to keep in mind if you want to host your own catalog.

  • Here is the repo that you can see all of the files.
  • In "Settings/Pages," pick a branch for the GitHub Pages source.
  • GitHub Pages takes time to rebuild the site, so making changes to the repo may take some time to show up on the website.
