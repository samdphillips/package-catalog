Preview Racket Package Catalog

[GitHub] This package catalog contains pre-release versions of my published packages and packages that I have not yet published into the main Racket Package Catalog.

How to use this catalog

$ raco pkg config --set catalogs \
    https://samdphillips.github.io/package-catalog/catalog/ \
    https://download.racket-lang.org/releases/$YOUR_RACKET_VERSION/catalog/   \
    https://pkgs.racket-lang.org \
    https://planet-compats.racket-lang.org
